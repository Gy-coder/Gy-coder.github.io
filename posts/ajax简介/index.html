Ajax简介

February 26, 2020

ajax是什么？

AJAX即“Asynchronous JavaScript and XML”（非同步的JavaScript与XML技术），指的是一套综合了多项技术的浏览器端网页开发技术。Ajax的概念由杰西·詹姆士·贾瑞特所提出

ajax的功能

  • ajax的功能就是用JavaScript来发送请求和接收响应

  • 浏览器在window上加了一个XMLHttpRequest函数

  • 用这个函数可以构造出一个对象

  • JavaScript可以用这个函数来发送请求和接收响应

ajax的用法

  1. 创建一个XMLHttpRequest对象

    const request = new XMLHttpRequest()

  2. 调用这个对象的open方法

    request.open('GET','/style.css')

  3. 监听对象的onreadystatechange事件

    request.onreadystatechange = ()=>{
    //具体代码
}

  4. 调用对象的send方法

    request.send()

关于open

  • open方法接收两个参数

    1. 第一个参数是是请求方式，有GET和POST两种
    2. 第二个参数是请求路径

关于onreadystatechange事件

  1. readystate

    XMLHttpRequest.readyState属性返回一个XMLHttpRequest代理当前所处的状态。一个XHR代理总是处于下列状态中的一个：

    值 状态 描述
    0 unsent 创建XMLHTTPRequest
    1 opened open方法调用
    2 headers-received send方法已被调用
    3 loading 下载中(第一个字符已被下载)
    4 done 下载操作完成

  2. status

    • status表示Http状态码

    • HTTP 响应状态代码指示特定HTTP请求是否已成功完成。响应分为五类：信息响应(100–199)，成功响应(200–299)，重定向(300–399)，客户端错误(400–499)和服务器错误 (500–599)

    • 一般请求成功为200，失败为404

    • 具体代码含义可以点击链接

ajax请求范例

  1. 请求CSS

    const request = new XMLHttpRequest()
request.open('GET','/style.css')
request.onreadystatechange =()=>{
   if(request.readystate === 4 && request.status === 200){
       const style = document.createElement('style')
       style.innerHTML = request.response
       document.head.appendChild(style)
   }
}
request.send()

  2. 请求JavaScript

    const request = new XMLHttpRequest()
request.open('GET','/javascript.js')
request.onreadystatechange =()=>{
   if(request.readystate === 4 && request.status === 200){
       const style = document.createElement('script')
       script.innerHTML = request.response
       document.body.appendChild(script)
   }
}
request.send()

  3. 请求HTML

    const request = new XMLHttpRequest()
request.open('GET','/other.html')
request.onreadystatechange =()=>{
   if(request.readystate === 4 && request.status === 200){
       const style = document.createElement('div')
       div.innerHTML = request.response
       document.body.appendChild(div)
   }
}
request.send()

  4. 请求XML

    const request = new XMLHttpRequest()
request.open('GET','/newxml.xml')
request.onreadystatechange =()=>{
   if(request.readystate === 4 && request.status === 200){
       const domobj = request.responseXML
       const text = domobj.getElementsByTagName('warning')[0].textContent
       console.log(text.trim())
   }
}
request.send()

  5. 请求JSON

    const request = new XMLHttpRequest()
request.open('GET','/newjson.json')
request.onreadystatechange =()=>{
   if(request.readystate === 4 && request.status === 200){
       const newObject = JSON.parse(request.response)
       console.log(newObject)
   }
}
request.send()

关于JSON

  1. JSON的数据类型

    1. string
    2. number
    3. bool
    4. null
    5. object
    6. array

    具体语法可以参考官方文档

  2. JSON中的方法

    1. parse方法

      • parse方法将符合JSON语法的字符串转化为JavaScript中对应类型的数据
      • 由于JSON的数据类型只有6种，所以转化的数据也只有6种
      • 如果不符合JSON语法，则直接抛出一个Error对象

    2. stringify方法

      • 是JSON.parse的逆运算
      • 由于JavaScript的数据类型比JSON多，所以不一定成功
      • 如果失败则抛出一个error对象
