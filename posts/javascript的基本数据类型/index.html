JavaScript的基本数据类型

December 9, 2019

JavaScript的七种数据类型

  • 数字类型 number
  • 字符串类型 String
  • 布尔类型 bool
  • 符号 Symbol
  • 空 undefined
  • 空 null
  • 对象 Object

JavaScript的数据类型可以总结为四基(数字、字符串、布尔、符号)两空(undefined、null)一对象(Object)

数据类型的详细介绍

  1. 数字类型number

    1. 写法

      1 //整数写法
0.1  //浮点数写法
1.23e4 //科学计数法，值为1.23 * (10^4)
0123  00123 0o123 //八进制写法
0xF1F  //十六进制写法
0b1011  //二进制写法

    2. 特殊值

      • 正0与负0:在一般情况下二者没有区别，但是在除0时，正负就出现作用了，如果是正0，则除法结果是正无穷，如果是负0，则除法的结果为负无穷

        1/0  //结果为+Infinity
1/-0 //结果为-Infinity

      • 无穷大:有 +Infinity 和 -Infinity 之分

    3. NaN:

      • NaN是指我们无法用现有的数学内容所表示出来的数字，他也是数字的一种
      • 关于NaN的一些注意事项
        1. 从bool逻辑上讲，NaN为false
        2. 从等于上讲NaN不等于任何东西 包括他本身(NaN === NaN的返回值为false)

  2. 字符串类型String

    1. 写法

      '你好'  //单引号写法
"你好"  //双引号写法
`你好`  //反引号写法

      但是假如字符串内本身有引号，则会出现歧义，如下所示

       'it's ok'  //浏览器会认为字符串为it

      为避免这样的状况出现，就需要转义字符能够让机器认识引号是字符串内容。

    2. 关于转义字符:

      • 转义的概念:用另一种写法表示你想要的内容
      • 转义的写法:在引号前在\则为对后面的一个引号转义

      • it’s ok的正确写法:

        'it\'s ok'

      • 常用的转义字符:

        \'   //表示'
\"   //表示"
\n   //表示换行
\r   //表示回车
\t   //表示制表符
\\   //表示\

    3. 多行字符串

      如果想在一个字符串内多行，用反引号写法很容易做到

        let s = `你好
      我是奥特曼`

    4. 字符串的属性:

      length方法:用于统计字符串的长度

       '123'.length  //串长为3
 '\n\t\r'.length  //串长为3
 ''.length  //串长为0
 ' ' .length  //串长为0

      注:空串和空格串的含义完全不通，空串长度为0，空格串长度为1

  3. bool类型

    1. 取值:bool类型只有两个值 true 和 false

    2. 如何得到bool值？

      通过如下三种计算

      !value  //否定运算
1 == 2   1 != 2 3 === 4 3 !== 4 //相等运算
1 > 2  1 >= 2 3 < 4 3 <= 4 //比较运算

    3. 常用的5个falsy值(重点、建议背下来)

      1. 0(数字0)
2. undefined
3. null
4. ''(空串)
5. NaN

  4. 关于undefined和null

    undefined和null没有本质区别，只有一些细节差异

    • 细节一:

      如果一个变量声明但是没有赋值，那么默认值为undefined而不是null

    • 细节二:

      如果一个函数没有写return，那么默认return undefined 而不是null

    • 细节三: 习惯上把非对象的空写为undefined，而把对象的空写为null

数据类型之间的转换

  1. Number ==> String

    1. String(n)
2. n + ''

  2. String ==> Number

    1. Number(s)
2. parseInt(s)
3. parseFloat(s)
4. s - 0

  3. X ==> bool

    1. Boolean(x)
2. !!x

  4. X ==> String

    1. String(x)
2. x.toString()

    注:toString方法有bug

    1.toString()  //会报错，因为Js认为1.之后应该接数字，所以报错
(1.).toString()  //正确

关于变量声明

变量声明有三种方式

var a = 1;
let a = 1;
const a = 1;

三种方式的区别:

  1. var是过时的方式，不推荐使用
  2. let是新的方式，推荐使用
  3. const方式必须对变量赋值且无法更改

let声明的特点:

  1. 遵循块作用域，范围不超过{}
  2. 变量不可重复声明
  3. 可以赋值也可以不赋值
  4. 必须先声明再使用，否则报错
  5. 全局声明的let变量不会成为window的属性

  6. for与let使用有奇效

    for(let i = 0; i < 5; i++){
    setTimeout(() =>{
        console.log(i);
    })
}
//打印结果为0 1 2 3 4 ，如果用var则会打印5个5
