Js函数的执行时机

January 10, 2020

一些小例子

例1

let a = 1
function fn(){
    console.log(a)
}

Q:打印多出少？

A:不知道，因为函数没有执行

例2

let a = 1
function fn(){
    console.log(a)
}
fn()

Q:打印多出少？

A:1

例3

let a = 1
function fn(){
    console.log(a)
}
a = 2
fn()

Q:打印多出少？

A:2

例4

let a = 1;
function fn(){
    console.log(a)
}
fn()
a = 2

Q:打印多出少？

A:1

例5

let a = 1
function fn(){
    setTimeout(()=>{
        console.log(a)
    },0)
}
a = 2
fn()

Q:打印多出少？

A:2

例6

let i
for(i = 0;i < 6;i++){
    setTimeout(()=>{
        console.log(i)
    },0)
}

Q:打印多出少？

A:6个6

setTimeout函数内的代码会在过一段时间后执行，而循环会一直执行，直到循环结束，SetTimeout中的代码执行，因此打印出6个6、

例7

for(let i = 0;i < 6;i++){
    setTimeout(()=>{
        console.log(i)
    },0)
}

Q:打印多出少？

A:0，1，2，3，4，5

当for和let联用时候，会在每一次循环时创建一个i，于是当循环结束时，每个setTimeout函数都会打印出自己的i，因此结果为0–5
