一些问题

  1. 如果异步任务有两个结果，成功或者失败。怎么办？

    方法一:回调接受两个参数

    fs.readFile('/1.txt',(error,data)=>{
    if(error){console.log("加载失败"),return }
    console.log(data.toString())  //成功
})

    方法二:搞两个回调

    ajax('get','/1.txt',data=>{},error=>{})
    //前面是请求成功的回调，后面是请求失败的回调

    ajax('get','/1.txt',{
    success:()=>{},
    fail:()=>{}
})
    //接受一个对象，有两个属性，分别表示成功和失败

  2. 这些方法的不足

    1. 不规范，名称五花八门
    2. 容易出现回调地狱，使代码可读性变差
    3. 很难进行错误处理

promise 解决这些不足的方法

  1. 以ajax的封装为例，解释promise的封装

  2. promise的封装和调用

    调用:

    ajax('get','/1.txt').then((resolve)=>{},(reject)=>{})

    封装:

    ajax = (method,url,options)=>{
    return new Promise((resolve,reject)=>{
    const {success,fail} = options
    const request = new XMLHttpRequest()
        request.open(method,url)
        request.onreadystatechange = ()=>{
            if(request.readystate === 4){
                if(request.status < 400){
                    resolve.call(null,request.response)
                }else if(request.status >= 400){
                    reject.call(null,request)
                }
        }
    }
    request.send()
    })
}

  3. 小结:

    1. 第一步:
      1. return new Promise((resolve,reject)=>{})
      2. 任务成功调用resolve
      3. 任务失败调用reject
      4. resolve和reje会再调用成功和失败函数
    2. 第二步:
      • 使用.then(success,fail)传入成功和失败函数
