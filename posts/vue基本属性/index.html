Vue基本属性

May 1, 2020

Vue的基本属性

  1. el–挂载点

    可以用$mount代替

  2. data–内部数据

    支持对象和函数，但是优先使用函数

  3. methods–方法

    事件处理函数或者是普通函数

  4. components–组件

    组件有三种写法

  5. 四个生命周期钩子

    • created –实例出现在内存中调用
    • mounted –实例出现在页面中调用
    • updated –实例更新时调用
    • destroyed –实例在内存中消亡时调用

  6. prop –外部数据

    • message = “n” –传入字符串
    • :message = “n” –传入this.n数据
    • :fn = “add” – 传入this.add函数

挂载点用法

  • 挂载点可以用el实现，也可以用$mount实现

    import App from './App.vue'
new Vue({
  el: "#app",
  render: h => h(App)
});

    import App from './App.vue'
new Vue({
  render: h => h(App)
}).$mount("#app");

data内部数据

  • data内部数据支持对象和支持函数，但是优先使用函数

    new Vue({
  el:"#app",
  template:`<div>这是第{{n}}个div</div>`,
  data(){
    return {n:0}
  }
})

methods用法

  • methods是一个对象

    new Vue({
    methods:{
       add(){
         this.n += 1
       }
    },
})

components组件用法

  • components组件有三种方法

  1. 单文件组件

    // 在App.vue文件中写入
<template>
  <div>{{n}}</div>
</template>

<script>
   export default {
       data(){
          return {n:0}
       },
       methods:{
          add(){
            this.n+=1
          }
       }   
   }
</script>

<style scoped>
       
</style>

    // 在main.js中写入
import App from './App.vue'
new Vue({
   el:"#app",
   component:{
      App:App
   },
   template:`<App/>`
})

  2. JavaScript内写入Vue组件

    //在main.js中写入
const Vue = window.Vue
Vue.component('Demo',{
    template:`
       <div>{{n}}</div>
    `,
    data(){
       return {n:0}
    }
})
new Vue({
   el:"#app",
   template:`<Demo/>` 
})

  3. component中写入组件

    const Vue = window.Vue
new Vue({
   el:"#app",
   component:{
      Demo:{
        template:`demo`
      }
   },
   template: `<Demo/>`
})

四个生命周期钩子

  1. created

    <script>
    export default {
        created(){
            console.log("在内存里了")
        },
    }
</script>

  2. mounted

    <script>
    export default {
        mounted(){
            console.log("在页面里了")
        },
    }
</script>

  3. updated

    <script>
    export default {
        updated(){
            console.log("更新了")
        },
    }
</script>

  4. destoryed

    <script>
    export default {
       destroyed(){
           console.log("没了")
       },
    }
</script>
