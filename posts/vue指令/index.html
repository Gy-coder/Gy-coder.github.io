May 20, 2020

Vue展示内容

  1. 表达式

    表达式内可以写表达式 如 {{n}}

    也可以写任何运算 如 {{n+1}}

    也可以调用函数 会返回函数执行的结果 如 {{fn(3,2)}}

    {{obj.a}}
{{n+1}}
{{fn(3,1)}}

    实例

  2. HTML内容

    假设data.x的内容为

    <strong>hi</strong>

    在template的div中写入

    <div v-html="x"></div>

    即可显示出粗体的hi

    实例

  3. 只显示{{n}}

    只需要在{{n}}的标签写入v-pre即可

     <div v-pre>{{n}}</div>

    实例

什么是指令？

  • 以 v- 为开头的东西就是指令

  • 指令的语法

    <div v-html="x"></div>

    v-指令名:参数=“值”

    <div v-on:click="add">123</div>

  • 关于指令

    1. 如果指令中没有特殊字符，可以不加引号

    2. 有些指令没有参数和值，比如v-pre

    3. 有些指令没有值，比如v-on:lick.prevent

主要的三个指令修饰符

  1. 阻止事件冒泡

    @click.stop="add"

.sync修饰符

Vue规则：

  1. 组件不能修改props外部数据

  2. $emit可以触发事件并传参

  3. $event可以获取$emit的参数

为了能够让组件修改props外部数据，我们有如下例子

在App.vue中

    <template>
      <div class="app">
        app.vue 有{{total}}
        <hr>
         <Child :money="total" v-on:update:money="total=$event"/> 
      </div>
    </template>
    
    <script>
        import Child from "./Child.vue";
        export default {
           data(){
             return {
             total:10000
           }
        },
        components:{Child}
       }
    </script>
    
    <style>
      .app{
        border:1px solid red;
        height: 100px;
      }
    </style>

在Child.vue中

    <template>
      <div class="child">
        {{money}}
        <button @click="$emit('update:money',money-100)">借钱</button>
      </div>
    </template>
    
    
    <script>
    export default {
      props: ["money"]
    };
    </script>
    
    
    <style>
    .child{
      border:1px solid blue;
    }
    </style>

在这个例子当中 ，Child组件可以修改App组件的数据，从而达到目标，为了使得Child标签更加简洁，可以简写为

   <Child :money.sync="total" />

.sync 修饰符只是作为一个编译时的语法糖存在。它会被扩展为一个自动更新父组件属性的 v-on 监听器
